# Advancement Multiplier

Every tracked advancement doubles your item drops. One advancement gives 2x, two give 4x. Get enough and you're mining Billions off a single block.

Includes Unlimited Stacks. No need to install both.

## How it works

Each advancement you earn doubles the multiplier. The current value shows on your bossbar and in chat, formatted as K/M/B/T/Q/Qi at higher values.

Root advancements (`story/root`, `nether/root`, `end/root`) are excluded.

## Installation

Download from [Modrinth](https://modrinth.com/mod/advancement-multiplier-mod).
